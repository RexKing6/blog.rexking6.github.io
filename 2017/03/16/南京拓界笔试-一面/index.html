南京拓界笔试+一面

发表于 2017-03-16 更新于 2018-03-25 分类于 笔试面试 阅读次数：
423 1 分钟

孝陵卫那里的路真不好，公交走起来都噼里啪啦听不清报站的声音。

今天去了拓界。

前天面完阿里，他们电话就打了过来，说在智联看到我简历，想约面试，本来以为是去面试的，没想到还要笔试。

他们公司是做信息取证的，大概就是帮政府在罪犯的存储设备中找到证据。

研发岗位主要是C++和Java，web开发的岗位貌似很少，可能也因为这样，没有运维岗吧。

因为我主要方向是运维开发和Python开发，再向大数据方面靠，这样方向有点对不上了。

笔试题很简单，唔……应该说太简单了，就比二级难了那么一丢丢吧，很多我了解到的C++开发中的东西都没有考到，包括在面试中也没有问到。

面试官问到的大概有Windows API，就说了下大二下做的输入输出框，提了下回调函数之类的，然后是指针和引用的差别，这个在好多地方看到过，就说了。

然后是链表和数组的差别，最后是双向链表的插入，感觉都好基础的说，基本都答出来了……

总之，他们说三天内会有消息，可能还有复试……但是方向不对，就有点不想去了……
